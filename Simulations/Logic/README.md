# Logic

These simulations are game-ified logic puzzles to help develop problem-solving and deductive reasoning skills.

- [Towers of Hanoi](TowersOfHanoi.html) - Solve the classic puzzle by moving all disks to the final peg without placing a larger disk on a smaller one. [2026-03-12 12:00:00]
- [Mastermind Codebreaker](Mastermind.html) - Use logic and deduction to break the hidden color code in 10 attempts or less. [2026-03-12 12:00:00]
- [Lights Out](LightsOut.html) - Turn off all the lights on the grid. Clicking a light toggles it and its adjacent neighbors. [2026-03-12 12:00:00]
- [Quantum Entanglement Puzzle](QuantumEntanglement.html) - A 4x4 grid puzzle where nodes are secretly entangled in pairs. Incrementing one decrements its hidden partner. [2026-03-13 22:09:06]
